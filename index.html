🔧 ¿Cómo configurar tu Micro:bit?

Para usar el guante pintor, debes cargar el código en tu Micro:bit para que envíe datos por Bluetooth.

📋 Paso 1: Copia el código

Elige el tipo de código que quieres usar:

# MakeCode Python (Recomendado - Conexión Dual USB/BLE)
conectado = False # Bandera para controlar el estado de la conexión inalámbrica

def on_bluetooth_connected(): # Evento que se dispara al vincular un celular/PC
    global conectado # Permite modificar la variable de estado
    conectado = True # Confirma la conexión exitosa
    basic.show_icon(IconNames.HAPPY) # Muestra cara feliz

def on_bluetooth_disconnected(): # Evento que se dispara al perder la vinculación
    global conectado # Permite modificar la variable de estado
    conectado = False # Confirma la pérdida de señal
    basic.show_icon(IconNames.ASLEEP) # Muestra cara dormida

bluetooth.on_bluetooth_connected(on_bluetooth_connected) # Activa el "escucha" de conexión
bluetooth.on_bluetooth_disconnected(on_bluetooth_disconnected) # Activa el "escucha" de desconexión

bluetooth.start_uart_service() # Enciende el servicio de envío de texto por Bluetooth
basic.show_icon(IconNames.ASLEEP) # Muestra el estado inicial

def on_forever(): # Bucle de ejecución continua
    x = input.acceleration(Dimension.X) # Lee la fuerza G en X
    y = input.acceleration(Dimension.Y) # Lee la fuerza G en Y
    z = input.acceleration(Dimension.Z) # Lee la fuerza G en Z

    btn_a = 1 if input.button_is_pressed(Button.A) else 0 # Convierte el estado del botón A a 1 o 0
    btn_b = 1 if input.button_is_pressed(Button.B) else 0 # Convierte el estado del botón B a 1 o 0

    msg = "" + str(x) + "," + str(y) + "," + str(z) + "," + str(btn_a) + "," + str(btn_b) # Crea una sola cadena de texto

    serial.write_line(msg) # VÍA 1 (Cable): Envía el texto directo al puerto COM

    if conectado: # Solo si hay un dispositivo vinculado...
        bluetooth.uart_write_line(msg) # VÍA 2 (Aire): Envía el texto a través de la antena Bluetooth

    basic.pause(30) # Pausa de seguridad para no saturar los puertos

basic.forever(on_forever) # Mantiene el código ejecutándose infinitamente
# MicroPython Estándar (Micro:bit V1/V2 - Conexión Dual)
from microbit import *
import bluetooth

uart = bluetooth.Uart()
uart.start()

display.show(Image.ASLEEP)

while True:
    x = accelerometer.get_x()
    y = accelerometer.get_y()
    z = accelerometer.get_z()
    btn_a = 1 if button_a.is_pressed() else 0
    btn_b = 1 if button_b.is_pressed() else 0

    msg = "{},{},{},{},{}".format(x, y, z, btn_a, btn_b)

    # Enviar por USB Serie
    print(msg)

    # Enviar por Bluetooth si está conectado
    if uart.is_connected():
        display.show(Image.HAPPY)
        uart.send(msg + "\n")
    else:
        display.show(Image.ASLEEP)

    sleep(30)
// MakeCode JS/Blocks (Conexión Dual USB/BLE)
let conectado = false
bluetooth.onBluetoothConnected(function () {
    conectado = true
    basic.showIcon(IconNames.Happy)
})
bluetooth.onBluetoothDisconnected(function () {
    conectado = false
    basic.showIcon(IconNames.Asleep)
})
bluetooth.startUartService()
basic.showIcon(IconNames.Asleep)

basic.forever(function () {
    let x = input.acceleration(Dimension.X)
    let y = input.acceleration(Dimension.Y)
    let z = input.acceleration(Dimension.Z)
    let btnA = input.buttonIsPressed(Button.A) ? 1 : 0
    let btnB = input.buttonIsPressed(Button.B) ? 1 : 0

    let msg = x + "," + y + "," + z + "," + btnA + "," + btnB

    // Enviar por USB Serie
    serial.writeLine(msg)

    // Enviar por Bluetooth
    if (conectado) {
        bluetooth.uartWriteLine(msg)
    }
    basic.pause(30)
})

🔗 Paso 2: Conectar

Una vez cargado el código en la Micro:bit, presiona el botón 📡 Conectar Bluetooth o 🔌 Conectar USB en la parte superior. ¡Se abrirá una ventana para emparejar tu dispositivo!

🎮 Paso 3: ¡A pintar!

Usa el Botón A para activar/desactivar el pincel, y el Botón B para calibrar. Mueve la Micro:bit para mover el cursor. ¡Diviértete!
